# Foxcc Internal

A free and open-source internal cheat for Counter-Strike 2, written in C++, working on Windows.

If you like this project, please give me a star :D

![image](https://github.com/jhl337/cs2_internal_foxcc/blob/master/showcase.png)

## Features

> 1. BoxESP
>
> 2. NameESP
>
> 3. HealthESP
>
> 4. AimBot (with RCS, team-check, vis-check, flash-check, and custom keybinds)
>
> 5. Triggerbot (with Delay and Duration to custom)
>
> 6. Radar (with team-check, Size, Proportion)
>
> 7. Rainbow Glow
>
> 8. No Flash
>
> 9. Spectator List
>
> 10. Safe Load and Unload

## Features that will be added in the future

>

>

>
> 4. Vis-check via map parsing

## Waring
The purpose of this project is to learn, please ignore the use for illegal purposes, otherwise at your own risk!

## How to use?
> 1. Download and import the "Minhook" and "ImGui".
>
> 2. Compile in "Release X64" mode.
>
> 3. Inject the dll to game (using Manual Map)
>

> 4. Waiting for load and enjoy!
